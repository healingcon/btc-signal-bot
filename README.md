## 🚀 설치 방법 (5분)

### 1단계 — 텔레그램 봇 만들기
1. 텔레그램에서 **@BotFather** 검색
2. `/newbot` 입력 → 봇 이름/ID 설정
3. `Bot Token` 복사해두기 (예: `7123456789:AAH...`)
4. 봇과 대화 시작 후 `https://api.telegram.org/bot<TOKEN>/getUpdates` 접속
5. `"chat":{"id":숫자}` 부분이 **Chat ID**

### 2단계 — GitHub 저장소 만들기
1. GitHub에서 **New Repository** 생성 (예: `btc-signal-bot`)
2. 이 파일들 전부 업로드

### 3단계 — Secrets 등록
저장소 → **Settings** → **Secrets and variables** → **Actions**

| 이름 | 값 | 구분 |
|------|-----|------|
| `TG_TOKEN` | 봇 토큰 | Secret |
| `TG_CHAT_ID` | 채팅 ID | Secret |

### 4단계 — 점수 임계값 설정 (선택)
Settings → **Variables** → `SCORE_THRESHOLD` = `70` (기본값)
- 70 이상: 롱 신호 / 30 이하: 숏 신호

### 5단계 — Actions 활성화 확인
저장소 → **Actions** 탭 → Enable 클릭

## ⚙️ 설정 변경

### 다른 코인 추가
`signal.yml`의 `SYMBOL` 부분 수정:
```yaml
SYMBOL: 'ETHUSDT'   # 이더리움
SYMBOL: 'SOLUSDT'   # 솔라나
```

### 실행 주기 변경
```yaml
# 30분마다
- cron: '*/30 * * * *'

# 매시간 정각
- cron: '0 * * * *'

# 한국시간 오전 9시 (UTC 0시)
- cron: '0 0 * * *'
```
